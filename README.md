# MarkDownEditer-tauri

js使えばjavaより楽そう

### 付けたい機能一覧
1. エクスプローラー
2. コードブロックの文字ハイライト
3. 上部のメニューバーの機能拡充
   - ファイル
     - 新しいマークダウンファイル
     - 新しい(その他の)ファイル
     - ファイルを開く
     - フォルダを開く
     - 保存
     - 名前を付けて保存
     - 設定
   - 編集
     - undo
     - redo
     - 切り取り
     - コピー
     - 貼り付け
     - 表・URL・画像の挿入
   - ヘルプ
     - マークダウン記法のチートシート
